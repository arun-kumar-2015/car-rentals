# **App Name**: Arun Car Rentals Online

## Core Features:

- Car Catalog: Display available cars with images, details (name, price, fuel type, seats, transmission), and a 'Book Now' button.
- Booking System: Allow users to select pickup/return dates, enter personal details (name, phone, license), and pickup location to calculate the total rental price.
- Firestore Integration: Store booking data (customer details, car, dates, total amount, status, timestamp) in Firestore under a 'bookings' collection.
- Admin Authentication: Implement Firebase Authentication for the admin panel, allowing login with email/password.
- Admin Panel: Enable admins to view all bookings, update booking status (Pending/Confirmed/Completed), and delete bookings.
- Contact Section: Provide contact information, including phone number, WhatsApp button, Google Maps embed, and a contact form.
- Responsive Design: Ensure the website is fully responsive and mobile-friendly, using smooth animations and a clean, modern UI.

## Style Guidelines:

- Primary color: Deep black (#121212) for a premium feel.
- Background color: Dark grey (#1E1E1E) provides contrast while maintaining a dark theme.
- Accent color: Bright yellow (#FFEA00) for highlights, buttons, and important information.
- Body and headline font: 'Inter' sans-serif for a modern and readable interface.
- Use professional, high-quality icons related to cars and rental services.
- Premium card design for car listings, with clear sections and a consistent grid layout.
- Smooth transitions and subtle animations to enhance the user experience.
